## quiz
设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

MinStack() 初始化堆栈对象。
void push(int val) 将元素val推入堆栈。
void pop() 删除堆栈顶部的元素。
int top() 获取堆栈顶部的元素。
int getMin() 获取堆栈中的最小元素。

## 思路
这题其实比有效的括号更简单，不知道为什么还标成中等  
首先这里最难的部分是在常数时间内获取最小值，所以我们肯定需要一个辅助栈  
然后在push的时候，比较push的val和主要栈的最顶处的大小，如果更小，那就放进辅助栈中，pop的时候，先pop出栈顶的值，然后比较pop出来的值和辅助栈栈顶的值，如果相等，那就一起pop出来。  
上面的思路说起来很简单，其实包含了很多栈的特定，比如我一开始想维护一个按照严格大小排序的辅助栈，但是实际上栈只能使用pop出栈。
举一个例子
3 1 2  
这时候的辅助栈为3 1
为什么2比3小不在辅助栈里面呢，因为2肯定会比3早出栈，不存在某一时刻，2是最小的时候。  
然后pop出栈也是同样的道理
